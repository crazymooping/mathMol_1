**เราคือหมอ** 🩺 ที่ต้องหาทาง **ฉีดวัคซีนให้ครบทุกบ้าน** โดยใช้เข็มให้น้อยที่สุด!

## 🔢 ตัวเลขบนบ้านคืออะไร?

แต่ละบ้านมี **ตัวอักษร 1 ตัว** อยู่ข้างใน เช่น `0` `3` `A` `F`

ตัวอักษรนี้คือ **"รหัสลับ"** ที่บอกว่า ชาวบ้านคนนั้น **กลัวใคร / รอใครฉีดก่อน**

### วิธีอ่านรหัสลับ

เอาตัวอักษรมา **แปลงเป็นเลข 0 กับ 1 จำนวน 4 ตัว** (เรียกว่า Binary):

```
ตัวอักษร → เลข 4 ตัว
               ↓  ↓  ↓  ↓
              บน ขวา ล่าง ซ้าย
```

**ถ้าเป็น 1** = ชาวบ้านคนนี้ **รอ** ให้เพื่อนบ้านทิศนั้นฉีดก่อน
**ถ้าเป็น 0** = **ไม่สน** ทิศนั้น

### ตัวอย่างง่ายๆ

| รหัส | แปลงเป็น | หมายความว่า |
|------|----------|-------------|
| `0` | `0 0 0 0` | 😄 **ไม่กลัวใครเลย!** ไม่ต้องรอใคร |
| `8` | `1 0 0 0` | 👆 **รอบ้านข้างบน** ฉีดก่อน ถึงจะยอมฉีด |
| `1` | `0 0 0 1` | 👈 **รอบ้านข้างซ้าย** ฉีดก่อน |
| `5` | `0 1 0 1` | 👉👈 **รอบ้านขวา + ซ้าย** ฉีดก่อน |
| `F` | `1 1 1 1` | 😰 **กลัวมาก! รอทุกทิศ** ฉีดก่อนหมด ถึงจะยอม |

> 💡 **เคล็ดลับ:** เอาเมาส์ไปชี้ที่บ้านในเกม จะเห็นเลข 4 ตัวนี้โผล่ขึ้นมาเลย!

## 🖱️ เล่นยังไง?

### ขั้นที่ 1: คลิกที่บ้าน
- **คลิก** ที่บ้านหลังไหนก็ได้ → บ้านนั้นจะ **โดนฉีดวัคซีนทันที!** 💉
- ทุกครั้งที่คลิก = ใช้เข็ม **1 เข็ม**

### ขั้นที่ 2: ดูปฏิกิริยาลูกโซ่!
- พอบ้านหลังนึงฉีดแล้ว → **บ้านข้างๆ จะเช็กตัวเอง**
- ถ้าบ้านไหน **"รอ" อยู่ แล้วคนที่รอฉีดแล้ว** → มันจะ **ยอมฉีดเองอัตโนมัติ!** ✨
- แล้วก็จะต่อเป็น **ลูกโซ่** ไปเรื่อยๆ ไม่เสียเข็มเพิ่มเลย!

### ตัวอย่าง:

```
บ้าน A อยู่มุมซ้ายบน   บ้าน B อยู่ข้างๆ
รหัส = 0 (ไม่รอใคร)    รหัส = 8 (รอบ้านบน)

1. เราคลิกบ้าน A → ฉีดแล้ว!  (ใช้ 1 เข็ม)
2. บ้าน B อยู่ข้างล่าง → "บ้านบน" ของ B คือ A
   → A ฉีดแล้ว! → B ยอมฉีดเอง! (ฟรี! ไม่เสียเข็ม)
```

> ⚠️ **ถ้าบ้านอยู่ขอบ** แล้ว "รอ" ทิศที่ไม่มีบ้านอยู่ (เช่น บ้านแถวบนสุดรอ "บน")
> → **ข้ามเงื่อนไขนั้นไปเลย!** ไม่ต้องสนใจ
